# Face-anti-spoofing
- Install
```
pip install -r requirements.txt
```
- Run `test.py`
```
python test.py --image_name 2605a340b6cd6b9332dc.jpg
```
- Train model
  - Format dataset
  - Run file `train.py` or run on google colab following file `train_face_anti.ipynb`
  
```
python train.py
```
